
# Ottonova Coding challenge
Socket.IO Chat Client with commands

## Requirements
- node (v16.8.9)ß
- npm  (v8.0.0)


## Installation

```
$ npm i
```

## Start application
```
$ npm start
```

## Load it in the browserß

Go to `http://localhost:3001`.
